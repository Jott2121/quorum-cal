# Run receipts

Reports, run logs, and panel configs for every measurement act (see WRITEUP.md
for the narrative). Gold sets here are FILTERED: 84 of the 180 phase2c tasks
(and the corresponding share of earlier sets) derive from a private repository
and are withheld from this public release. All aggregate numbers in the
reports include the withheld tasks. The published tooling rebuilds an
equivalent gold set from any repository with a real test suite. Judgment
caches are not committed (they are large and re-runnable); smoke caches are
committed as receipts.
