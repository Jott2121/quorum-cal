# Run receipts

Reports, run logs, and panel configs for every measurement act (see WRITEUP.md
for the narrative; report-v1.0.0.json preserves the pre-correction phase2c
numbers). Gold sets here are FILTERED: 84 of the 180 phase2c tasks (and the
corresponding share of earlier sets) derive from a private repository and are
withheld from this public release. All aggregate numbers in the reports
include the withheld tasks. The published tooling rebuilds an equivalent gold
set from any repository with a real test suite. Judgment caches are not
committed (large, re-runnable); smoke caches are committed as receipts.
